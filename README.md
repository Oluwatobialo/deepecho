
# DeepEcho Clinical SaaS Platform

This is a code bundle for DeepEcho Clinical SaaS Platform. The original project is available at https://www.figma.com/design/XuQUj9yK9sMNc0VS6lk5BC/DeepEcho-Clinical-SaaS-Platform.

## Running the code

- **Backend:** Requires **Python 3.11 or 3.12** (see [START.md](START.md)). Run from project root: **run-backend.bat** (Windows) or from `backend`: `python -m uvicorn main:app --host 127.0.0.1 --port 8002`.
- **Frontend:** Run `npm i` then `npm run dev` (or **run-frontend.bat**) to start the development server.
